Para executar seu projeto em modo tela cheia no Scratch, vá para a área acima do Palco e clique no ícone com quatro setas que apontam para fora. Este é o ícone **Controle de Tela Cheia**:

![O ícone 'Controle de Tela Cheia' destacado acima do Palco, no canto direito.](images/fullscreen_frank.png)

Para sair do modo tela cheia, clique no ícone **Controle de Tela Cheia** outra vez. Ele terá quatro setas que apontam para dentro.
